Instagram Clone Project by AADITYA JAISWAL 
A full-stack social media platform inspired by Instagram, built using the MERN stack (MongoDB, Express, React, Node.js). This project allows users to create posts, follow others, like/dislike content, comment, and personalize their profiles.


Installation


Backend
1.	Navigate to the backend folder:
             cd backend
2.	Install dependencies:
              npm install
3.	 Start the server:
Npm start
 Frontend
1.	Navigate to the frontend folder:
             cd frontend
2.	Install dependencies:
              npm install
3.	Start the React app:
              npm start


Features


User Authentication
•	Users can sign up and log in securely.
•	Token-based authentication to protect private routes.
Profile Management
•	Users can update personal details like gender, date of birth, and bio.
•	Upload profile pictures (display pictures) to Cloudinary.
Posts
•	Create, edit, and delete posts with images.
•	View a feed of all posts.
•	Like and dislike posts.
•	Save posts to bookmarks.
Comments
•	Add comments on posts.
•	Edit or delete comments (only by the owner of the comment).
•	View all comments on a post.
Follow System
•	Follow and unfollow other users.
•	View users not followed yet.
Other Features
•	Tag creation (for posts).
•	Responsive and clean UI.
•	Error handling with feedback using toast notifications.
Tech Stack
Frontend:
•	React.js
•	React Router DOM for navigation
•	Axios for API calls
•	React Hot Toast for notifications
Backend:
•	Node.js with Express.js
•	MongoDB with Mongoose
•	Cloudinary for image storage
•	JWT for authentication
